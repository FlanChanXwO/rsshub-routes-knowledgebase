往需要数亿甚至数十亿投入，绝大多数中小企业难以承受。目前大型科研仪器设备、大科学装置等关键资源主要集中在高校和科研院所，开放共享率偏低，企业获取研究条件的渠道有限、成本较高。另一方面，基础研究成果产出周期长，不确定性大，与企业追求短期经济效益的目标存在天然矛盾，企业开展基础研究的内生动力不足、顾虑较多。</p><p>企业缺位是我国基础研究体系的最大结构性短板。对企业自身而言，基础研究是构建长期核心竞争力的根本保障，只有掌握底层原理和核心技术，才能摆脱技术依赖，在全球产业链分工中占据主动地位。企业也是连接基础研究与产业应用的关键桥梁，企业参与基础研究能够有效解决科研与产业“两张皮”问题，大幅提升成果转化效率，让基础研究成果更快转化为现实生产力。因此，<strong>必须把激发企业基础研究活力作为完善我国基础研究投入体系的核心任务之一。</strong></p><p><strong>三、构建支持基础研究的多元投入保障体系</strong></p><p>加快构建全方位、多层次的基础研究支持保障体系，不仅需要中央财政持续发力，还需要进一步发挥地方财政和特大企业的积极作用，形成中央统筹引领、地方协同参与、社会资本补充联动的多元化投融资体系。</p><p><strong>（一）建立中央财政主导、地方财政配套的基本投入格局</strong></p><p>基础研究事关国家整体利益、覆盖全国公共利益，属于典型全国性公共产品，必须强化中央财政主体责任，发挥主导和压舱石作用。要建立中央财政基础研究投入稳定增长机制，确保投入规模持续扩大、增速稳步提升，匹配基础研究长周期、高投入、慢回报特征，为基础研究提供长期、稳定、可预期的资金支持。</p><p>基础研究能够支撑地方产业升级、人才集聚，带动地方科技水平和创新能力，地方是基础研究的直接受益者。<strong>合适的激励有助于释放地方政府的参与积极性。</strong>如对于地方政府投入基础研究的资金，中央财政可考虑给予一定比例的配套支持，撬动地方加大投入；还可以支持发达地区设立区域性基础研究基金，整合区域内科技资源，开展跨区域协同攻关，提升区域基础研究整体水平。考核激励机制的优化也能够实现有效引导，如将基础研究投入占比、重大原创成果产出等指标纳入地方政府绩效考核体系。</p><p>夯实基础研究条件支撑体系是中央和地方财政投入的重点。重大科技基础设施是开展高水平基础研究的必备条件。对于投资规模大、建设周期长的国家级重大科技基础设施，可以通过发行特别国债或者专项债等方式筹集建设资金，缓解各级财政压力。</p><p><strong>（二）积极利用各类财政金融工具，充分激发企业积极性</strong></p><p>企业是技术创新的主体，也是基础研究的重要力量，充分激发企业投入积极性是完善多元投入体系的关键抓手。为鼓励企业直接开展基础研究，可根据基础研究投入占企业R&#x26;D比重给予累进式财政补贴和税收优惠，基础研究投入占比越高，财政补贴返还比例越高；完善科技创新融资担保配套机制，对基础研究投入占比越高的企业，适度放宽担保授信额度、下调担保费率，降低企业基础研究资金筹措成本。</p><p><strong>整体上，企业直接开展基础研究的成本高、周期长、风险较为集中，应当进一步创新企业参与方式。</strong>鼓励和支持企业与高校、科研院所共建联合实验室、产业技术研究院等创新平台，围绕产业关键底层技术开展协同攻关，形成“企业出题、高校解题、成果共享、风险共担”的产学研融合模式。为了解决仪器设备紧缺问题，可以推广设备租赁模式，由政府或第三方机构建设大型科研仪器共享平台，企业以租赁方式使用仪器设备，能够有效降低企业固定资产投入。对企业用于联合实验室建设、设备租赁、课题研究等基础研究相关的支出，可进一步提高计算应纳税所得额时的加计扣除比例。</p><p>依托政府投资基金可以进一步撬动社会资本投身基础研究领域。如设立国家级企业基础研究引导基金，重点扶持具备产业转化潜力的应用基础研究，以及有望催生颠覆性技术的前沿科研探索。通过引导基金示范带动，鼓励地方政府设立配套基金、吸引社会资本参与，共同支持重大科技基础设施建设、前沿基础研究项目落地，形成政府引导、市场运作、多元投入的基础研究投融资局面。</p><p><strong>（作者席鹏辉为中国社会科学院财经战略研究院研究员、财政研究室副主任，研究领域为财税理论与政策，近著《知难而进：全面深化改革下的地方财政》由商务印书馆出版）</strong></p><img height=\"1280\"><p>“李时珍1518蕲艾香灸”包装信息。</p><p></p><p></p><p>近日，澎湃新闻前往湖北省襄阳市和黄冈市蕲春县，探访“李时珍1518蕲艾香灸”包装上标注的生产厂家和地址，但均未找到实际生产地。6月10日，襄阳市樊城区市场监管局工作人员向澎湃新闻表示，正在调查此事，但无法提供具体情况。此外，襄阳市樊城区烟草专卖局一名工作人员表示，消费者如认为是类烟产品，可以通过国家烟草专卖局互联网售烟信息举报小程序进行反映。</p><p><strong>网店销售香炙称可像香烟一样吸食，专家：对身体有害</strong></p><p>山东聊城市民徐先生近日向澎湃公众互动平台“服务湃”（https://tousu.thepaper.cn）反映称，今年4月5日，他花费10600元一次性网购20盒“李时珍1518蕲艾香灸” 【注：李时珍为蕲州（今蕲春县）人，生于1518年】。该产品宣称具备养生保健、祛湿排寒、润肺、清除血管垃圾等多重功效。但使用一周后，徐先生出现嗓子发痒、疼痛等不适症状。他向商家申请十倍赔偿，尚未得到支持。</p><img style=\"max-width: 600px; height: auto; display: block; margin: 0 auto;\" src=\"https://imgpai.thepaper.cn/newpai/image/1781181630247_i7eN4K_1781181670736.jpg?x-oss-process=image/resize,w_1024\" alt=\"\" width=\"1290\" height=\"2620\"><p>“李时珍1518蕲艾香灸”在各大电商平台上仍有售卖。</p><p></p><p></p><p>徐先生发给记者的商品页面截图显示，“李时珍1518蕲艾香灸” 产品不含尼古丁，内含艾草及中草药，具有养生保健、祛湿排寒、润肺、清除血管垃圾等多重功效，宣称可作为“香烟平替”。此外，页面还宣称“香炙全身都是宝”，“炙灰滴到鱼缸，快死掉的鱼有可能让它复活”。徐先生说，该产品从外观到使用方式均与普通卷烟完全一致，“有过滤嘴，也是点燃吸食，抽烟怎么抽，这个就怎么抽”。</p><p>澎湃新闻在淘宝、抖音等电商平台上检索发现，同款“李时珍1518蕲艾香灸”均有售卖，且销量不低。以淘宝“李时珍蕲艾香炙工厂店”为例，该店铺单盒（20支装）售价66元，单条（10盒装）售价540元，“全网1000+人付款”。商品页面仅有制作工艺和作用等宣传信息，未标注使用方法。</p><p>重庆一家三甲医院呼吸内科主治医师黄医生介绍，吸食艾草制品对身体是有害的。从使用方式看，它类似香烟，它与香烟危害健康的原理是一样的，艾草、草本植物燃烧同样会产生颗粒物（PM2.5）、一氧化碳、焦油、多环芳烃等有毒有害物，烟雾被直接吸入呼吸道，会刺激咽喉、气管、肺部，诱发干咳、咽干、气道炎症，长期吸入会损伤肺组织，增加呼吸系统疾病风险；另外，人参、虫草、藏红花、三七等原料仅为草本成分，经高温燃烧后有效成分大多被破坏、分解，不会随烟雾被人体吸收，且肺部生理结构并非为处理燃烧烟雾而进化的，反而药材燃烧会额外产生复杂杂环化合物，进一步加重呼吸道、心肺负担。黄晓寒说，中药成分复杂，传统艾灸是皮肤局部温热刺激+少量吸入，且通常在通风环境，这样的方式会有大量的烟雾吸入肺部，还有中毒风险。</p><p>对于商家宣传的功效，黄医生表示，呼吸道吸入烟雾只会伤肺、加重肺燥、刺激气道，不存在“润肺”作用；祛湿、散寒是中医外治（艾灸、热敷、泡脚）的作用，吸入烟雾无法实现体内祛湿排寒，且传统中医治疗方法中并无吸入疗法；黄晓寒说，现代医学没有“血管垃圾”这一正规概念，烟雾中的有害物质反而会损伤血管内皮、影响血脂代谢，只会伤害血管，绝无清理作用。</p><p><strong>探访：厂家注册地及生产地址均“查无此厂”</strong></p><p>徐先生提供的产品包装信息显示，“李时珍1518蕲艾香灸”由湖北中艾农业集团有限公司（下称：中艾公司）出品。生产地址位于湖北省黄冈市蕲春县漕河镇李时珍医药港。主要成分包含人参（人工种植）、蕲艾、虫草、藏红花、三七等，产品执行标准为Q/ZAJT002-2025。但包装上未标识生产批号等重要信息。</p><img alt=\"\" width=\"2260\" 伙同刘某向隋某等人销售。报道称，中医概念上的“香灸”是用中草药制成的条或柱，用火点燃，在经络上进行治疗而达到祛病止痛和保健功效等目的。仲某等人掩盖产品的烟草特性，通过虚假宣传，诱骗大量不明真相的群众上当受骗，不仅扰乱烟草市场秩序，更损害消费者身体健康。针对新型涉烟产品属性模糊、监管依据不足等问题，2024年3月，山东省莱州市检察院在深入调查后，向莱州市烟草专卖局制发检察建议，建议其加大对网络售烟、类烟制品的打击力度，进一步明确混合中药、改变形态后的类烟制品定性等。</p><p>此外，还有商户询问是否可以销售茶烟等类烟产品，烟草主管部门答复称：茶烟等类烟产品往往经过“戒烟”“好抽不上瘾”等噱头的包装营销，极易诱导热衷网络、好奇心强、自制力弱的青少年群体接触。同时，类烟产品缺乏强制标准，存在产品用料不明等质量安全问题。此外，类烟产品在生产经营中往往会使用卷烟纸、滤嘴棒、烟用丝束、烟机等烟草专卖品，严重违反《烟草专卖法》《烟草专卖法实施条例》等相关法律法规，还有大量类烟产品假冒知名卷烟商标或标识，误导广大消费者购买吸食，会对零售商家的信誉造成损失。</p><p>黄医生亦提醒，点燃吸食艾草或草本烟雾会对人体呼吸系统、心血管系统造成明确伤害，这类产品所宣传的祛湿、润肺、清血管等功效都属于虚假夸大宣传，没有科学与医学依据支撑，同时仿香烟式吸食草本制品，已被多地市场监管部门判定为违规产品，不建议任何人使用，大家切勿轻信、购买以及使用。若想要借助艾草进行身体调理，应当选用艾灸、泡脚、煮水外用等传统安全外用方式，日常养生调理最好前往正规医院，在专业中医师的指导下进行，远离所有吸食类“养生烟”产品。</p><img
