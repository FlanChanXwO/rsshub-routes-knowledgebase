认证、授权使用、收益分配和安全审计等方面加强规则衔接。依托可信数据空间建设，探索企业、行业、城市、跨境等多类型应用场景，同步推广隐私计算、联邦学习等“可用不可见”技术，降低数据流通中的安全顾虑和合规成本。只有让数据可用、可信、可交易、可增值，让资金、人才、技术与数据在区域内高效聚合，长三角才能真正把资源优势转化为配置优势和发展优势。</p><p><strong>（二）一体化统筹发力，把“区域优势”转为“整体优势”</strong></p><p>长三角最大的特点，不只是产业强、城市密、市场大，更在于其本质上是高度网络化、强联系的区域共同体。数智化转型如果停留在城市、园区和平台之间“各自为战”，就会削弱一体化优势。下一步，应从分散建设转向一体化统筹，从行政推动转向规则共建，从重规模扩张转向重使用效能。特别是在工业互联网、算力基础设施、公共数据服务和跨区域场景应用等领域，要围绕“能否互联、能否调度、能否共享、能否协同”建立统一标准和运行机制，强化芜湖算力平台等核心载体的跨域调度能力，推动区域算力资源协同优化配置。只有把标准体系、平台体系和调度体系真正打通，长三角企业数智化转型才可能从局部领先走向整体跃升。</p><p><strong>（三）场景牵引分层赋能，把“先发优势”转为“全域成效”</strong></p><p>长三角企业数智化转型最终能否落地，不取决于少数龙头企业走得多快，而取决于广大制造业企业特别是中小企业能否真正受益。应避免把数智化转型讲成“大而全”的口号，也不能简单等同于设备更新、软件采购和平台上线。对多数制造业企业而言，更务实的路径是从最紧迫、最能见效的业务场景切入，先在研发设计、生产管控、供应链协同、质量追溯、节能降碳等环节形成突破，再逐步向全流程延伸。对龙头企业，应鼓励其开放平台、场景和标准，发挥链主带动作用；对中小企业，则应提供轻量化改造、低成本服务、诊断评估、融资增信和人才培训支持，帮助其以渐进方式融入区域数智化进程。只有让更多企业看到收益、降低顾虑、增强能力，长三角的先发优势才可能转化为区域范围内的普遍成效。</p><p><strong>（本文作者曾刚系华东师范大学城市发展研究院院长、教育部人文社会科学重点研究基地中国现代城市研究中心主任，杨创豪系华东师范大学中国现代城市研究中心/地理科学学院博士研究生，陈鹏鑫系华东师范大学城市发展研究院博士后、助理研究员）</strong></p><p><span style=\"max-width: 600px; height: auto; display: block; margin: 0 auto;\" src=\"https://imgpai.thepaper.cn/newpai/image/1780502262369_Rba27X_1780502286855.png?x-oss-process=image/resize,w_1024\" alt=\"\" width=\"1915\" 较琐碎，有时候不好做，这些年你有遇到过特别难的事吗？</p><p><strong>向金元</strong>：难的事肯定有。基层工作第一要有奉献，第二要有情怀。大家都是农村出身的，如果你带着真情实感去做事，办法总比困难多。当然，哪里有一马平川走到底的？也有解决不了的问题，但大多数能够解决。</p><p>总有磕磕坎坎，就看你怎么去克服。</p><p><strong>澎湃新闻</strong>：你这种格局和处事态度，是性格决定的，还是工作中历练的？</p><p><strong>向金元</strong>：性格有一部分影响，工作中也历练了一部分。</p><p><strong>澎湃新闻</strong>：以前遇到特别困难的事时，有没有不想干了的时候？</p><p><strong>向金元</strong>：有。但有时候一脚踏进来了，就像开弓没有回头箭。遇到这种情况，有的人可能会选择逃避。你不能逃避现实，只能勇往直前，不能临阵脱逃。</p><p>我是一名共产党员，我的责任和使命所在，危难当头必须挺身而出。</p><p><strong>澎湃新闻</strong>：让你坚持下来的动力是什么？因为你是党员？</p><p><strong>向金元</strong>：这是肯定的。当初是党员和人民群众把你选出来的，危难当中你自己退缩，像只缩头乌龟，你从良心上对不起老百姓。人家把希望寄托在你身上，你总得拿一份工资做一份事吧。</p>,与澎湃新闻记者对话时，前不久因耳环遭遇网暴的女村支书向金元身心已经完全平静。
